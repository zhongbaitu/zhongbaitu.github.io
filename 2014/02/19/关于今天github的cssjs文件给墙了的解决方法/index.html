关于今天Git Hub的Css/Js文件给墙了的解决方法

By 钟买能

Feb 19 2014 Updated:Apr 5 2014

Contents

今天早上一打开 github 发现 css 等都加载失败。

后来发现是给墙了。

本人用的是 windows。

解决方法：改 hosts

一般情况是在这里：
C:\Windows\System32\drivers\etc\hosts

打开。在后面加上这句

185.31.16.184 github.global.ssl.fastly.net

OK。可以正常访问了。

Mac OS X 的在这里：/private/etc

翻墙的方法有好多种。强悍的程序员们也会很多。刚群上有个哥们问了这事情。每个人的知识面都不一样，还是有人不知道的。所以特此分享一下。

ps：facebook、google+ 等网站都可以用这个方法翻墙去浏览的哦。若还需知更多信息可以咨询谷歌大哥。
